Le Démineur

Les règles

But du jeu

Il faut révéler toutes les cases de la grille sans jamais retourner l'une de celles qui cache une mine.

Comment jouer ?

Faire un clic gauche sur une case révèle ce qu'elle cache.

Faire un clic droit permet de poser un drapeau sur une case où l'on suspecte la présence d'une mine.

Les différentes cases

Les mines : si vous en révélez une, vous perdez la partie.

Les chiffres : ils indiquent combien de mines se trouvent parmi les 8 cases voisines.

Les cases vides : en réveler une permet de révéler toutes les cases vides qui sont adjacentes.
